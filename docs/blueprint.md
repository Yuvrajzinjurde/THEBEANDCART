# **App Name**: AuthFlow

## Core Features:

- Sign Up: Allow new users to create an account with an email address and password.
- Login: Enable existing users to log in with their credentials.
- Password Reset: Provide a way for users to reset their password if they forget it, with email verification.
- Form validation: Validate input fields to provide real-time error reporting
- AI Enhanced Password: Employ an AI tool to measure password strength as users create new passwords, and advise about improving it.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and security.
- Background color: Light indigo (#E8EAF6), a heavily desaturated variant of the primary, creates a clean backdrop.
- Accent color: Vivid violet (#7C4DFF) for interactive elements and error highlighting, drawing attention while staying harmonious.
- Font: 'Inter', a grotesque sans-serif for a modern, neutral, and readable feel across the app.
- Use a centered layout with clear sections for login and signup forms to ensure user-friendliness and intuitive navigation.
- Incorporate subtle transitions and animations to provide visual feedback and enhance the overall user experience during authentication.
